# Ch01 — 時間序列探索性資料分析

## 學習目標

- 理解時間序列資料的基本結構與特性
- 學會使用分解法拆解趨勢、季節性、殘差
- 掌握定態性檢定（ADF、KPSS）的意義與判讀
- 識別自相關（ACF）與偏自相關（PACF）模式

## 核心觀念

**時間序列分析三要素**：
```
原始序列 = 趨勢 (Trend) + 季節性 (Seasonality) + 殘差 (Residual)
```

**定態性 (Stationarity)** — 統計性質不隨時間改變，是多數統計模型的前提假設。
