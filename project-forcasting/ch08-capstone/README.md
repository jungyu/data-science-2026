# Ch08 — 綜合專案：端對端預測系統

## 學習目標

- 整合前七章所學，建構完整的預測管線
- 實作多模型 Ensemble（投票法/堆疊法）
- 產出可復現的實驗報告
- 建立模型選擇與部署的決策框架

## 專案要求

使用本專案學到的技術，完成以下任務：

### 基礎要求
1. 選擇一個預測標的（股票、ETF、加密貨幣等）
2. 實作至少 3 種不同方法的預測模型
3. 使用 Walk-forward 驗證評估所有模型
4. 含交易成本的回測分析
5. 使用 MLflow 記錄所有實驗

### 進階挑戰
- 實作 Ensemble（Voting/Stacking）
- 多股票/多市場預測
- 加入外部資料（總經指標、新聞情緒）
- 建立自動化預測排程
