## 教學資料集

本手冊全程使用一套模擬的**台灣電商營運資料庫**作為教學素材：

```
docs/pandas/data/
├── orders.csv        # 訂單資料（核心主表）
├── customers.csv     # 顧客資料
├── products.csv      # 商品資料
├── campaigns.csv     # 行銷活動資料
├── returns.csv       # 退貨資料
└── inventory.csv     # 庫存資料
```

> 這不是隨便湊的假資料。每張表都**刻意設計**了：
> - 缺失值（讓你練 `fillna`）
> - 異常值（讓你練偵測與處理）
> - 重複資料（讓你練 `drop_duplicates`）
> - 日期格式不一致（讓你練 `to_datetime`）
> - 跨表關聯（讓你練 `merge`）

詳見 [data/README.md](data/README.md)

## 怎麼使用這本手冊

### 如果你是學生 🎓

1. **按順序讀**：Ch01 → Ch11，不要跳章
2. **每章都動手做**：光看不練，等於沒學
3. **用自己的資料試**：學完一章，就拿自己的 CSV 練一次
4. **寫學習筆記**：用自己的話重新解釋每個概念

### 如果你是老師 👨‍🏫

1. 參考 [附錄 A：教學指南](appendix-teaching-guide.md) 的 18 週課程設計
2. 每章的「教學重點」區塊標示了核心觀念
3. 「常見錯誤」區塊可作為課堂討論素材
4. 資料集已內建教學用的髒資料，可直接使用

### 如果你是自學者 🚀

1. 先跑一遍 Ch10 的完整專案，感受全貌
2. 回頭從 Ch01 開始扎實學習
3. 遇到不懂的，善用 `help()` 和官方文件
4. 完成後挑戰：用 Pandas 分析一個你關心的真實資料集
