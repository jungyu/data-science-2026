# 《進階資料科學方法論》— 六週學生模組

> 讓學生從「會跑模型」進階為「能設計決策系統的人」。

---

## 模組總目標

完成本模組後，你應能：

1. 理解數據中的不確定性
2. 設計一個可驗證的實驗
3. 評估模型是否真的有價值
4. 辨識常見的統計誤用
5. 用方法論框架思考任何資料問題

---

## 每週教材

| 週次 | 主題 | 核心問題 |
|------|------|----------|
| [Week 1](week-01-uncertainty.md) | 不確定性的本質 — 樣本與母體 | 你看到的數字，能代表真相嗎？ |
| [Week 2](week-02-hypothesis-testing.md) | 假設檢定與其陷阱 | p < 0.05 就代表有效嗎？ |
| [Week 3](week-03-causal-inference.md) | 因果推論與實驗設計 | 相關真的等於因果嗎？ |
| [Week 4](week-04-model-training.md) | 模型訓練與資料切分 | 模型在訓練集上表現好，就夠了嗎？ |
| [Week 5](week-05-model-evaluation.md) | 模型評估與指標選擇 | Accuracy 高就是好模型嗎？ |
| [Week 6](week-06-integration.md) | 整合 — 從數據到決策 | 怎麼把分析變成可行動的建議？ |

---

## 模組核心思維

請記住三句話：

1. **不被樣本欺騙** — 任何數字都有不確定性
2. **不把相關當因果** — 沒有實驗設計，就沒有因果宣稱
3. **不被模型指標迷惑** — 指標要配合情境選擇，沒有萬用指標
