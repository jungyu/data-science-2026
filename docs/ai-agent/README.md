## 本手冊的教學哲學

### 🧠 深入淺出原則

本手冊借鏡 O'Reilly「Head First」系列的教學設計：

1. **故事先行** — 每章以真實場景開場，讓你先感受「為什麼這很重要」
2. **視覺化思考** — 大量圖示、流程圖、對照表，讓抽象概念具體化
3. **動手做** — 每章都有可執行的練習，不只是「看懂」而是「會用」
4. **對話式學習** — 用「如果你問我...」的方式回答常見疑問
5. **漸進式挑戰** — 從簡單到複雜，每一步都建立在前一步之上

### 📐 三層理解模型

```
Level 3  ┌─────────────────┐
能教人    │  設計自己的框架  │  ← Ch13 畢業專案
         ├─────────────────┤
Level 2  │  靈活應用到專案  │  ← Ch05-Ch12 實戰與治理
會用      ├─────────────────┤
Level 1  │  理解核心概念    │  ← Ch01-Ch04 基礎篇
看懂      └─────────────────┘
```
